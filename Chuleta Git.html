⎇ Chuleta de Git

los comandos esenciales, en una página ← volver a la guía

Empezar

git initcrea un repositorio aquí
git clone <url>descarga un repo existente
git status¿cómo están mis archivos?
git log --onelinehistorial de commits

Guardar trabajo

git add <archivo>prepara un archivo (staging)
git add .prepara todo lo cambiado
git commit -m "msg"guarda la foto en el historial
git commit --amendrehace el último commit (local)

Sincronizar

git pullbaja e integra lo del remoto
git pushsube tus commits al remoto
git fetchbaja sin integrar (solo mirar)
git remote add origin <url>conecta con la nube

Ramas

git branchlista las ramas
git switch <rama>cámbiate de rama
git switch -c <rama>crea y cámbiate
git merge <rama>fusiona esa rama en la actual

Deshacer (local)

git restore <archivo>descarta cambios del archivo
git restore --staged <a>sácalo del staging
git reset --soft HEAD~1borra commit, conserva cambios
git reset --hard HEAD~1borra commit y cambios ⚠

Deshacer (publicado)

git revert HEADcommit nuevo que deshace el último
git revert <hash>deshace un commit concreto
⚠ nunca reset/amendsobre commits ya pusheados
